Tables

Bootstrap formats tables with .table and adds a light padding and horizontal dividers.
You can add .table-striped in order to give the table zebra stripes.
You can add .table-bordered in order to add borders on a sides of the table and cells
You can add .table-dark in order to add a dark theme to the table. The .table-hover class enables a hover state (grey background on mouse over) on table rows:
